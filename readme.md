# Proposal-Type-A-MC

## 시작하기
### 설치
- `yarn install` ( `npm install` )
### 개발 환경 서버실행
- `yarn dev` ( `npm run dev` )
### 빌드
- `yarn build` ( `npm run build` )

## 환경
- `gulp` 와 `webpack` 으로 환경이 셋팅 되어 있습니다.
- `gulp` 에서 `html, less, images, image sprite` 를 처리 합니다.
- `webpack` 에서 `javascript` 를 처리 합니다.
